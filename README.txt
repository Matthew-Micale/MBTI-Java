How to Download and Run Program

1. Download Eclipse at https://www.eclipse.org/downloads/
2. Launch Eclipse
3. Open File menu
4. Choose import
5. Choose Projects from Git (smart import)
6. Copy this URL: https://github.com/Matthew-Micale/MBTI-Java.git
7. Enter GitHub login
8. Confirm the directory to clone the project
9. Run the driver.java file
